##  About Me

Software Engineer with a background in **Applied Mathematics and Computer Science**, working across **systems programming, developer tooling, and production infrastructure**.

I build performance-critical and long-lived software, primarily in C++, spanning graphics and simulation pipelines, engine-style applications, internal developer tools, and platform-level systems. Im comfortable working close to the hardware, but I optimise for correctness, clarity, and maintainability over cleverness.

I pair low-level engineering with practical backend and infrastructure work, often using small, purpose-built services and APIs to support native tools and pipelines. I take systems from design through build, packaging, deployment, and long-term operation.

**Focus:** systems programming, graphics & simulation, developer tooling  
**Core skills:** C++, Python, TypeScript 路 Vulkan/OpenGL, SDL, Dear ImGui 路 Linux, system design 路 CMake & Conan, CI/CD, Docker 路 PostgreSQL & SQLite
